WHAT THIS PACKAGE CONTAINS
==========================
This package of software from the Essential Project contains the following:
- essential_baseline_v62.pprj	= The Protege project configuration for the repository
- essential_baseline_v62.pins 	= The Protege instances for the Essential repository
- essential_baseline_v62.pont	= The Protege ontology that describes the Essential Meta Model

These 3 files are inter-dependent and must be used together. 

GETTING STARTED
===============
To start using the Essential Architecture Manager:
- Start Protege
- Open essential_baseline_v612.pprj (using File->Open...)
- You may wish to save a backup at any time. Do this using File->Save As... in Protege


INSTALLATION
============
These files are the Essential Architecture Manager repository and we recommend that you install it in your filesystem in an easy-to-manage location, e.g. '/EssentialAM/Repository'
